# 📡 네트워크 접속 내역 - 3대 핵심 기능 설명 (PPT 작성용)

---

## 1. 🔴 실시간 모니터링

### 📋 기능 개요
**실시간 네트워크 통신 감시 시스템**
- 컴퓨터에서 발생하는 모든 네트워크 통신을 실시간으로 감시하는 기능
- 마치 네트워크 교통상황을 실시간으로 보는 것과 같음
- 네트워크의 "지금 이 순간" 상태를 즉시 파악

### 🔧 주요 기능
#### **1) 실시간 패킷 캡처**
- 들어오고 나가는 모든 네트워크 데이터를 즉시 포착
- 밀리초 단위의 실시간 반응성
- 네트워크 인터페이스별 선택적 모니터링

#### **2) 스마트 필터링 시스템**
- **IP 주소 필터**: 특정 IP로부터의 통신만 골라보기
- **프로토콜 필터**: TCP/UDP/ICMP 등 원하는 프로토콜만 표시
- **실시간 검색**: 입력과 동시에 즉시 필터링 적용

#### **3) 프로세스 추적**
- 어떤 프로그램이 네트워크를 사용하는지 실시간 확인 (PID, 작업 관리자에서 확인 가능)

#### **4) 대안 모니터링 시스템**
- 패킷 캡처가 불가능한 환경에서도 작동
- Windows 이벤트 로그 활용한 실시간 감지
- 중복 제거 시스템으로 정확한 데이터 표시
- 이중화 모니터링으로 누락 없는 추적

### 📊 표시 정보
- **시간**: 패킷 발생 정확한 시각
- **소스/대상 IP**: 통신 출발지와 목적지
- **포트 번호**: 사용된 네트워크 포트
- **프로토콜**: 통신 방식 (TCP, UDP, HTTP 등)
- **프로세스**: 네트워크를 사용한 프로그램명
- **패킷 크기**: 전송된 데이터 크기
- **방향**: 송신/수신/내부 통신 구분

### 👥 사용자 혜택
- **즉시 파악**: 현재 무엇이 인터넷에 연결되어 있는지 한눈에 확인
- **보안 강화**: 의심스러운 연결이나 프로그램을 즉시 발견 가능
- **문제 해결**: 네트워크 문제 발생 시 원인을 실시간으로 추적
- **성능 모니터링**: 네트워크 사용량이 많은 프로그램 식별

## 2. 📊 사용 기록

### 📋 기능 개요
**네트워크 사용 패턴 분석 시스템**
- 과거 네트워크 사용 내역을 체계적으로 분석하고 조회하는 기능
- "언제, 어디서, 무엇을" 연결했는지에 대한 기록

### 🔧 주요 기능
#### **1) 다중 소스 데이터 수집**
- **Windows 이벤트 로그**: 방화벽 허용/차단 기록 분석
- **netstat 명령어**: 현재 활성 연결 상태 실시간 수집
- **WMI 시스템**: 네트워크 인터페이스별 상세 통계

#### **2) 기간별 조회 시스템**
- **유연한 날짜 범위**: 원하는 시작일~종료일 자유 설정

#### **3) 고급 필터링 옵션**
- **인터페이스별**: Wi-Fi, 이더넷 등 네트워크 카드별 구분
- **프로토콜별**: TCP, UDP, ICMP, HTTP, HTTPS 구분
- **방향별**: 송신/수신/내부 통신 분류

#### **4) 실시간 통계 대시보드**
- **총 연결 수**: 선택 기간 동안의 전체 네트워크 연결 횟수
- **총 데이터량**: 송수신된 전체 데이터 크기 (자동 단위 변환: B/KB/MB/GB)
- **주요 프로토콜**: 가장 많이 사용된 통신 방식 분석
- **피크 시간**: 네트워크 사용이 가장 활발했던 시간대 식별

### 📈 제공 통계
#### **연결 분석**
- 시간대별 연결 패턴 분석
- 요일별 네트워크 사용 트렌드
- 프로토콜별 사용량 비율


### 👥 사용자 혜택
- **패턴 분석**: 네트워크 사용 패턴 분석으로 최적화 방안 도출
- **문제 추적**: 특정 시점의 네트워크 문제 원인 역추적

### 사용 기록이 원하는대로 출력되지 않는 이유?
- **이벤트 로그 보존 정책** 
    Windows 기본값: Security 로그 크기 제한 (일반적으로 20MB)
    자동 덮어쓰기: 로그가 가득 차면 오래된 이벤트가 자동 삭제
    보존 기간: 시스템 설정에 따라 며칠~몇 주만 보존
- **감사 정책 활성화 시점**
    활성화 이전의 네트워크 활동은 로그에 기록되지 않음

## 3. 🛡️ 보안 모니터링

### 📋 기능 개요
**네트워크 보안 위협 탐지 시스템**
- 네트워크 트래픽에서 보안 위협을 자동으로 감지하고 분석하는 지능형 보안 시스템
- 사이버 공격을 사전에 차단하는 예방적 보안 솔루션
- 전문 지식 없이도 누구나 사용할 수 있는 자동화된 보안 분석

### 🔧 핵심 보안 탐지 기능
#### **1) 악성 IP 감지**
- **블랙리스트 검사**: 알려진 해커 IP나 악성 사이트와의 통신 탐지
- **지리적 위험도**: 고위험 국가에서의 접속 시도 감지

#### **2) 의심스러운 포트 모니터링**
- **해킹 포트**: 해킹에 자주 사용되는 포트 접근 감지
- **서비스 포트**: 중요 서비스 포트의 무단 접근 탐지
- **비표준 포트**: 정상적이지 않은 포트 사용 패턴 분석
- **포트 스캔**: 연속적인 포트 탐색 시도 감지

#### **3) 비정상 트래픽 패턴 분석**
- **연결 빈도**: 평소와 다른 이상한 연결 횟수 감지
- **시간 패턴**: 비정상적인 시간대의 네트워크 활동
- **데이터 패턴**: 일반적이지 않은 데이터 전송 패턴
- **프로토콜 이상**: 비정상적인 프로토콜 사용 감지

#### **4) 대용량 데이터 전송 감지**
- **정보 유출**: 의심스러운 대용량 데이터 업로드 감지
- **임계값 설정**: 정상 범위를 벗어난 데이터 전송량 알림
- **시간대 분석**: 업무시간 외 대용량 전송 감지
- **목적지 분석**: 외부로의 대용량 전송 위험도 평가

### ⚠️ 위험도 분류 시스템
#### **🔴 심각 (Critical)**
- 즉시 조치 필요한 심각한 보안 위협
- 시스템 침해 가능성이 매우 높은 상황
- 자동 알림 및 긴급 조치 권고

#### **🟠 높음 (High)**
- 빠른 대응이 필요한 중요 위협
- 보안 정책 위반 가능성
- 상세 분석 및 조치 계획 수립 필요

#### **🟡 보통 (Medium)**
- 주의 깊게 관찰이 필요한 위험
- 잠재적 보안 취약점
- 정기적 모니터링 및 예방 조치

#### **🟢 낮음 (Low)**
- 일반적인 보안 이벤트
- 정상 범위 내의 활동
- 기록 및 통계 목적

### 📊 보안 통계 대시보드
#### **실시간 보안 현황**
- **총 경고 수**: 발견된 전체 보안 위협 건수
- **24시간 경고**: 최근 하루 동안 발생한 새로운 위협
- **고위험 경고**: 즉시 대응이 필요한 심각한 위협 수
- **미해결 경고**: 아직 처리되지 않은 보안 이슈 수

#### **위협 유형별 분석**
- 악성 IP 접근 시도 횟수
- 의심스러운 포트 스캔 감지 건수
- 비정상 트래픽 패턴 발견 횟수
- 대용량 데이터 전송 감지 건수

#### **조치 권고**
- 위협별 구체적인 대응 방안 제시
- 단계적 보안 조치 가이드
- 예방을 위한 권고사항

### 👥 사용자 혜택
- **사전 예방**: 해킹 시도나 악성코드 감염을 조기에 발견
- **정보 보호**: 개인정보 유출이나 데이터 탈취 위험 사전 차단
- **쉬운 사용**: 전문 지식 없이도 보안 상태를 쉽게 파악
- **자동화**: 복잡한 보안 분석을 자동으로 수행
- **비용 절약**: 별도의 값비싼 보안 솔루션 구매 없이 기본 보안 확보

## 🌟 통합 시너지 효과

### 💡 3가지 기능의 조합 이점
#### **연속적 보안 커버리지**
- **실시간** → **기록** → **보안**: 현재 상황 파악 → 과거 분석 → 미래 위협 예방
- **시간축 완전 커버**: 과거-현재-미래의 모든 네트워크 활동 추적
- **사각지대 제거**: 어떤 네트워크 활동도 놓치지 않는 완전한 모니터링

#### **포괄적 네트워크 가시성**
- **수평적 가시성**: 모든 네트워크 인터페이스와 프로토콜 커버
- **수직적 가시성**: 물리 계층부터 응용 계층까지 전 영역 분석

### 🎯 대상 사용자별 활용 방안
#### **IT 관리자 및 보안 담당자**
- 기업 네트워크 보안 강화
- 컴플라이언스 요구사항 충족
- 인시던트 대응 및 포렌식 분석

#### **개인 사용자 (개인정보 보호 관심자)**
- 개인 컴퓨터 보안 상태 확인
- 의심스러운 프로그램 활동 감지
- 개인정보 유출 위험 사전 차단

#### **소규모 기업의 네트워크 관리 담당자**
- 비용 효율적인 네트워크 보안 솔루션
- 전문 인력 없이도 가능한 보안 관리
- 네트워크 성능 최적화 지원

#### **사이버 보안에 관심이 있는 일반 사용자**
- 네트워크 보안 지식 습득
- 실제 네트워크 동작 원리 이해
- 보안 의식 향상

### 🚀 핵심 경쟁 우위
#### **실시간성**
- 밀리초 단위의 즉각적인 반응
- 위협 발생 즉시 감지 및 알림
- 실시간 의사결정 지원

#### **정확성**
- 중복 제거 알고리즘으로 정확한 데이터
- 다중 소스 검증으로 신뢰성 확보
- False Positive 최소화

#### **편의성**
- 직관적이고 이해하기 쉬운 UI
- 자동화된 분석 및 알림
- 전문 지식 없이도 사용 가능

#### **경제성**
- 별도 하드웨어 없이 소프트웨어만으로 구현
- 기존 Windows 기능 활용으로 비용 절약
- 오픈소스 기반으로 투명성 확보

## 📝 결론

이 네트워크 접속 내역 기능은 **실시간 모니터링**, **사용 기록 분석**, **보안 위협 탐지**의 3대 핵심 기능을 통해 
사용자에게 완전하고 안전한 네트워크 환경을 제공합니다.

모든 계층에서 활용할 수 있는 차세대 네트워크 보안 솔루션입니다.

**"보이지 않는 네트워크 위협을 보이게 만드는 혁신적인 솔루션"**
