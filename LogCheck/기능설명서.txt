===============================================================================
                    Windows Sentinel - LogCheck 기능 설명서
===============================================================================

프로젝트 개요:
Windows Sentinel은 Windows 시스템의 보안 모니터링 및 네트워크 분석을 위한 
종합적인 WPF 애플리케이션입니다.

개발 환경:
- 플랫폼: .NET 8.0 WPF
- 언어: C#
- UI 프레임워크: Material Design in XAML
- 차트 라이브러리: LiveCharts2
- 패킷 캡처: SharpPcap

===============================================================================
                              주요 기능 모듈
===============================================================================

1. 메인 대시보드 (MainWindow)
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • 시스템 개요 및 상태 모니터링                                          │
   │ • 실시간 시스템 리소스 사용량 표시                                      │
   │ • 보안 경고 및 알림 센터                                                │
   │ • 각 모듈로의 빠른 네비게이션                                           │
   │ • 다크/라이트 테마 지원                                                 │
   └─────────────────────────────────────────────────────────────────────────┘

2. 네트워크 모니터링 (Network.xaml)
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ 2-1. 실시간 패킷 캡처                                                   │
   │     • 네트워크 인터페이스 자동 감지 및 선택                             │
   │     • 실시간 패킷 캡처 및 분석                                          │
   │     • 프로토콜별 필터링 (TCP, UDP, ICMP, HTTP, HTTPS)                   │
   │     • 패킷 상세 정보 표시 (IP, 포트, 크기, 플래그)                      │
   │                                                                         │
   │ 2-2. 네트워크 사용 기록                                                 │
   │     • 날짜 범위별 네트워크 활동 조회                                     │
   │     • 인터페이스별, 프로토콜별, 방향별 필터링                           │
   │     • 실시간 통계 대시보드:                                             │
   │       - 총 연결 수                                                      │
   │       - 총 데이터 전송량                                                │
   │       - 주요 프로토콜                                                   │
   │       - 피크 시간대                                                     │
   │                                                                         │
   │ 2-3. 고급 차트 시각화                                                   │
   │     • 차트 기간: 시간별(24시간), 일별(7일), 주별(4주)                   │
   │     • 차트 유형: 연결 수, 데이터량, 프로토콜별 사용량                   │
   │     • 지능형 데이터 처리:                                               │
   │       - 최소 데이터 포인트 보장                                         │
   │       - 시간별 트래픽 패턴 시뮬레이션                                   │
   │       - 연속적 시간 범위 표시                                           │
   │     • 실시간 차트 업데이트                                              │
   │                                                                         │
   │ 2-4. 보안 모니터링                                                      │
   │     • 네트워크 위협 실시간 감지                                         │
   │     • 보안 경고 분류 및 관리                                            │
   │     • 위험도별 필터링 (낮음, 보통, 높음, 심각)                          │
   │     • 보안 통계 대시보드                                                │
   └─────────────────────────────────────────────────────────────────────────┘

3. 설치된 프로그램 관리 (InstalledPrograms.xaml)
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • 시스템에 설치된 모든 프로그램 목록 표시                               │
   │ • 프로그램 정보: 이름, 버전, 설치 날짜, 크기                           │
   │ • 검색 및 필터링 기능                                                   │
   │ • 프로그램 제거 기능 (관리자 권한 필요)                                 │
   │ • 설치 프로그램 통계 및 분석                                            │
   └─────────────────────────────────────────────────────────────────────────┘

===============================================================================
                              시스템 요구사항
===============================================================================

운영체제: Windows 10/11 (관리자 권한 필요)
.NET 버전: .NET 8.0 Runtime
추가 요구사항:
  • Npcap (패킷 캡처용)
  • Visual C++ Redistributable
  • 최소 4GB RAM 권장
  • 네트워크 어댑터 접근 권한

===============================================================================
                              사용 방법
===============================================================================

1. 애플리케이션 시작
   • 관리자 권한으로 실행 필수
   • 초기 설정 자동 진행 (감사 정책, Npcap 확인)

2. 네트워크 모니터링
   • 네트워크 탭 선택
   • 원하는 인터페이스 선택 후 캡처 시작
   • 실시간 패킷 분석 및 필터링

3. 사용 기록 분석
   • 사용 기록 탭 선택
   • 날짜 범위 및 필터 조건 설정
   • 조회 버튼 클릭하여 데이터 분석

4. 차트 분석
   • 차트 기간 및 유형 선택
   • 실시간 업데이트되는 시각화 데이터 확인
   • 통계 정보 활용

===============================================================================
                              문제 해결
===============================================================================

일반적인 문제:
1. "관리자 권한 필요" 오류
   → 마우스 우클릭 → "관리자 권한으로 실행" 선택

2. 네트워크 캡처 실패
   → Npcap 설치 확인
   → 네트워크 어댑터 드라이버 업데이트

3. 차트가 표시되지 않음
   → 데이터 필터 조건 확인
   → 테스트 데이터 생성 버튼 활용

4. 프로그램 목록 오류
   → Windows 레지스트리 권한 확인
   → 시스템 재시작 후 재시도
