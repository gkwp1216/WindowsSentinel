개인 PC에서 포트와 보안
1. 기본 개념
	클라이언트 PC(개인용) → 보통은 외부에 서비스 제공을 안 함 → “열린 포트(listening)”가 거의 없어야 정상.
	사용자 PC가 외부 서버에 접속할 때 → 로컬 임시 포트(ephemeral port)를 사용 (49152~65535 범위).
	만약 특정 서비스 포트(예: 21, 23, 3389 등)가 열려 있다면 → 불필요한 서비스 실행 중이거나 악성 프로그램일 수 있음.

2. 개인 PC 보안에서의 포트 관리 원칙
	불필요한 서비스 비활성화
		(예: 원격 데스크톱(RDP, 3389)을 쓰지 않으면 꺼두기.)
	방화벽 활성화
		Windows Defender 방화벽, macOS 방화벽, Linux ufw 등을 켜두기.
	암호화된 프로토콜만 사용
		HTTP 대신 HTTPS, FTP 대신 SFTP 등.
	정기적으로 확인
		netstat, ss, lsof 명령어 또는 보안 프로그램으로 열린 포트 점검.
